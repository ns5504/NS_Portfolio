US Alternative Fuel & EV Adoption Dashboard

Excel • SQL • Tableau • Data Visualization • Policy Analytics

Problem

Support transportation planning by understanding how EVs and alternative fuels are adopted across U.S. states, and identify where charging infrastructure investment should be prioritized.

My approach

  • Cleaned and standardized state-level registration data across fuel types.
  • Calculated EV adoption rate (% of total vehicles) for fair comparisons.
  • Ranked top and bottom states by adoption and examined regional patterns.
  • Built an interactive Tableau dashboard for non-technical stakeholders.

Key findings

  • EV adoption is concentrated in a small set of coastal/urban states.
  • California leads by both volume and adoption rate.
  • E85 is the only alternative fuel with broad national presence.
  • Hydrogen vehicles remain niche and concentrated in California.
  • Several states lag, indicating infrastructure/policy gaps.

Analyzing & Validating the Migration of Scientific Data at NSLS-II

Python • SQL • NoSQL → SQL Migration • Data Validation • Logging • Visualization

Overview

At Brookhaven National Laboratory’s National Synchrotron Light Source II (NSLS-II), I validated the migration of 10+ petabytes of scientific data from MongoDB (NoSQL) to PostgreSQL (SQL) within the Tiled data management framework.

My role

  • Validated correctness of large-scale NoSQL → SQL migration.
  • Compared query outputs across legacy and new systems.
  • Monitored run counts, metadata consistency, and migration logs.
  • Built visual summaries to communicate migration health.

Methods

  • Compared daily run counts (2021–2025) across systems.
  • Verified run-type distributions (FlyPlan 1D, FlyPlan 2D, others).
  • Parsed logs and categorized warnings vs errors.
  • Visualized trends using Matplotlib.

Key findings

  • Run-type distributions matched within <0.5%.
  • 64.5% of runs completed with no issues.
  • ~35% completed with warnings but no data loss.
  • Only 0.1% failed, indicating strong reliability.

Why this matters

Confirmed the SQL-backed system preserves data integrity at scale, supporting faster access, improved reliability, and a future facility-wide rollout across NSLS-II beamlines.

Assessing the Impact of ChatGPT in Education (NLP)

Python • Reddit Data • Sentiment Analysis • Topic Modeling • Model Evaluation

Overview

Analyzed 2,086 Reddit posts from education-focused subreddits (Dec 2022–Aug 2024) to understand perceptions of ChatGPT’s impact on learning, teaching, and academic integrity.

Methods

  • Collected posts via PullPush API (keyword “ChatGPT”) across multiple subreddits.
  • Cleaned spam/duplicates, filtered to English, removed incomplete entries.
  • Created a validation set of 200 labeled posts with refined guidelines.
  • Sentiment: VADER, BERT, RoBERTa, Logistic Regression + GPT comparisons.
  • Topics: LDA, CorEx, BERTopic.

Key findings

  • Strong concern around plagiarism and cheating.
  • Fear of reduced critical thinking due to over-reliance on AI.
  • Educators reported difficulty assessing authentic learning.
  • Topics surfaced ethics, unequal access, and policy uncertainty.

Model insights

  • Outputs varied by model (manual labels skewed more negative than GPT).
  • GPT-3.5/GPT-4 with reasoning handled nuanced cases more consistently.

Luxury Fashion Retail Performance Dashboard

Excel • Tableau • Data Cleaning • Luxury Retail Analytics • Dashboarding

Goal

Analyze sales performance, pricing discipline, seasonality, and return behavior across categories and brands to support data-driven merchandising, planning, and operational decision-making in a luxury retail environment.

What I did

  • Conducted rigorous data cleaning and validation in Excel, standardizing key fields including season, category, and return status.
  • Developed structured helper metrics (markdown amount, return flags, season order) and built pivot-based analyses.
  • Evaluated category-, brand-, and season-level performance to identify commercial priorities.
  • Assessed markdown behavior to understand pricing strategy and margin implications.
  • Analyzed return rates and return drivers to highlight potential risks to customer experience and operational efficiency.

Key insights

  • Outerwear emerged as the highest-revenue category, supporting its prioritization in assortment and allocation strategies.
  • Shoes, Dresses, and Bottoms showed the highest markdown intensity, while Outerwear maintained the strongest pricing discipline.
  • Summer generated peak sales, with Zara leading overall and across Winter and Spring, Mango strongest in Fall, and H&M in Summer.
  • Bottoms and Outerwear recorded the highest return rates, signaling areas for improved fit consistency and expectation management.
  • “Changed Mind” and “Size Issue” were the dominant return reasons, suggesting that enhanced product presentation and sizing clarity could reduce returns and operational costs.
